Specifications overview
The specifications contained in this section are intended to enable apps to
provide for and properly support all appropriate Roku devices.
This section contains the following types of information:
Hardware specifications — Device model code names,
model-family numbers,
and associated hardware characteristics (e.g., CPU and graphics device,
system clock speed, onboard RAM and flash memory, maximum video resolution for
UI and video playback, highest level of WiFi support, supported video output
modes, power requirements, and many more);
Streaming specifications — Video and audio file
formats and streaming protocols;
DRM & content protections — Guidelines
and recommendations for stream
encoding and DRM;
Roku Search feed format — Specifications for
formatting a Roku Search
content feed, which enables content from an app to be discoverable from
Roku's universal search feature;
Streaming Store graphics — Format details for
implementation of standard graphic elements of an app's UI.
